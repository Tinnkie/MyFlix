MyFlix Movies Library

As a user you will be able to receive information on available movies, directors, and genres so you can learn more about the movies you've watched or you are interested in.

You will be able to create a personal profile, and add or remove a movie from your list of favourites.

Endpoint Documentation

Business Logic URL HTTP Method Query Parameters Request Body Data Format Response Body Data Format
Returns a list of all movies /movies GET - - A JSON object holding data about all the movies
Returns data about single movie by title /movies/:title GET :title - A JSON object holding data about a single movie
Returns data about a genre by title /movies/genre/:genreName GET :genreName - A JSON object holding data about a specific genre of movie
Returns data about a director by name /movies/directors/:directorName GET :directorName - A JSON object holding data about a specific director
Allows users to add a movie to the database /movies POST - A JSON object holding data about the movie to add, structured like: { "Title": "String","Description": "String"} A JSON object holding data about the movie added, structured like: { "Title: "String, "Description": "String"}
Allows new users to register /users POST - A JSON object holding data about the user to add, structured like: { "Username": "String","Password": "String","Email": "String","Birthday": "Date" } A JSON object holding data about the user that was added, structured like: { "Username" : "Name1", "Password": "abcd", "Email": "example@gmail.com", "Birthday": "1990-05-01", "Favorite Movie" : [] }
Allows users to update their user name /users/:id PUT :id - A JSON object holding data about the updated user information
Allows users to add a movie to their favorites /users/:id/:movieTitle POST :id :MovieTitle - A JSON object holding data about the user and the movie that was added
Allows users to remove a movie from their favorites /users/:id/:movieTitle DELETE :id :MovieTitle - A JSON object holding data about the user and the movie that was removed
Allows existing users to deregister /users/:id DELETE :id - A text message indicating whether the user has successfully deregistered
