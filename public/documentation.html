MyFlix Movies Library

As a user you will be able to receive information on available movies, directors, and genres so you can learn more about the movies you've watched or you are interested in.

You will be able to create a personal profile, and add or remove a movie from your list of favourites.

Endpoint Documentation

Method Endpoint Description Request Body Response
POST /users Create a new user JSON object with Username, Password, Email, and Birthday JSON object with user data
GET /users Get all users None Array of JSON objects with user data
GET /users/:Username Get a user by username None JSON object with user data
PUT /users/:username Update a user by username JSON object with Username, Password, Email, and Birthday JSON object with updated user data
DELETE /users/:Username Delete a user by username None Success or error message
POST /users/:username/movies/:movieId Add a movie to user's favorites None JSON object with updated user data
DELETE /users/:id/movies/:MovieId Delete a movie from user's favorites None JSON object with updated user data
GET /movies/:title Retrieves a Movie from the database by its title None JSON Movie object
GET /movies/genre/:genreName Retrieves a Genre from the database None JSON Genre object
GET /movies/directors/:directorName Retrieves a Director from the database None JSON Director object
